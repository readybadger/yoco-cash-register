## Get started
1. Install dependencies
   ```bash
   pnpm install
   ```

2. Start the app
   ```bash
    npx expo run:android
     # or
    npx expo run:ios
   ```

## Features
- Fully typed with Typescript, linted and formatted with Prettier
- Settings page, including:
    - Theme (Dark/Light) mode
    - Toggle for auto-scrolling the charge list
    - Currency selector
